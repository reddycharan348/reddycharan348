# Portfolio Website for E Charan Kumar Reddy

This is a modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features
- **Modern UI**: Dark theme with neon accents and glassmorphism effects.
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Animations**: Smooth scroll and fade-in effects.
- **Sections**: Home, About, Projects, Services, Contact.

## How to Run
Simply open the `index.html` file in your web browser.

## Customization
- Edit `index.html` to update text content.
- Edit `styles.css` to change colors (check the `:root` variables).
- Edit `script.js` for behavior changes.
